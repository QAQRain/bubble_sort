# Bubble Sort

氣泡排序法 (Bubble Sort) 的 Python 實現。

## 什麼是氣泡排序？

氣泡排序是最基礎的排序演算法之一，它的原理是：
- 比較相鄰兩個元素，如果順序錯誤就交換它們
- 重複這個過程直到整個陣列排序完成
- 每輪排序會將最大的元素「冒泡」到正確的位置

## 時間複雜度

- 平均/最差：O(n²)
- 最佳：O(n) - 當陣列已經排序時

## 功能

- `bubble_sort(arr)` - 將陣列進行氣泡排序

## 使用方式

```python
from bubble_sort import bubble_sort

data = [64, 34, 25, 12, 22, 11, 90]
sorted_data = bubble_sort(data)
print(sorted_data)  # [11, 12, 22, 25, 34, 64, 90]
```

## 執行範例

```bash
python bubble_sort.py
```

輸出：
```
原始陣列: [64, 34, 25, 12, 22, 11, 90]
排序後: [11, 12, 22, 25, 34, 64, 90]
```

## 演算法說明

1. 第一輪：從第一個元素開始，比較相鄰元素並交換直到最大的元素移到最後
2. 第二輪：排除最後一個元素，對前面 n-1 個元素重複同樣動作
3.以此類推，直到所有元素排序完成
